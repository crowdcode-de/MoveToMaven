MoveToMaven-CLI Version 1.0.0-SNAPSHOT
--------------------------------------
You can run the movmvn-cli with following parameters:
- Parameter 1 = -zip or -dirProject or -dirProjectGroup
- Parameter 2 = zip name or directory name or directory group name
- Parameter 3 = working directory
		
Example with directory group name:
movmvn-cli -dirProjectGroup D:/progjava/movmvn-transformer-cli-1.0.0-SNAPSHOT/examples/root-directory D:/progjava/movmvn-transformer-cli-1.0.0-SNAPSHOT/examples
